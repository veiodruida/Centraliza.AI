# Centraliza.ai v0.3.1 - UX/UI Hardening Release

This release focuses on visual stability, accessibility, and user education.

## ✨ Highlights
- **Redesigned Dashboard**: New layout with Quick-Start Guide and FAQ.
- **Extreme Legibility**: Larger fonts and high-contrast for technical paths.
- **Smart Adaptive Buttons**: Fully responsive .btn-premium component.
- **Engine-Specific Filtering**: Compatibility checks in the Test Chat.

- **Layout Polish**: Full model names visible (no more line-clamping).
