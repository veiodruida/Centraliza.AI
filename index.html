FAQ

Will it break my LM Studio models?

No. Centraliza.ai uses safe symlinks that tools like LM Studio see as normal files. Your existing workflows remain intact.

Does it download models?

Yes. Via the AI Model Hub, you can search and download models directly from Ollama and HuggingFace repositories.

Is it free?

Yes. Centraliza.ai is an open-source tool built to empower the local AI community.
